### Цель и задачи

* Опробовать типовой стек команд **GNS3** на практике.

### Ход работы

<Subtitle>
  ## Часть 1 — Развёртывание инфраструктуры
</Subtitle>

1. Запустите лабораторную работу, нажав справа на кнопку **"Запустить"**


2. Разместите оборудование.

   * 2 × компьютера (PC)

3. **Соедините устройства** витой парой (Ethernet0/0, FastEthernet0/1 и т. д.) в соответствии с диаграммой.

4. **Назначьте IP-адреса** компьютерам
   Пример для PC1:

```bash
ip 192.168.1.10 255.255.255.0 192.168.1.1
```

5. **Проверьте link-local связность**:
```bash
PC1> ping 192.168.1.1
```

# Основы компьютерных сетей

Компьютерные сети – это неотъемлемая часть современной цифровой инфраструктуры. Благодаря сетям мы можем передавать данные между устройствами, взаимодействовать в глобальном масштабе, обеспечивать доступ к сервисам и информации из любой точки мира. В этом курсе мы рассмотрим основные концепции, протоколы и технологии, на которых строится современный интернет и корпоративные сети.

<Callout className="mt-6">
**Главная идея**: Компьютерные сети – это совокупность связанных между собой устройств и технологий, позволяющих обмениваться данными.
</Callout>

## Ключевые понятия

- **Узел (Host)**: Любое устройство, подключённое к сети (компьютер, сервер, принтер и т. д.).
- **Протокол**: Набор правил и соглашений, определяющих порядок взаимодействия между устройствами в сети.
- **Среда передачи**: Физический (кабель, оптоволокно) или беспроводной (Wi-Fi, Bluetooth) канал, по которому передаются данные.
- **Трафик**: Поток данных, циркулирующих между узлами.

## Основные преимущества сетей

1. **Обмен информацией**: Возможность быстрого и удобного обмена файлами и сообщениями.
2. **Совместное использование ресурсов**: Общий доступ к принтерам, серверам и другим устройствам.
3. **Масштабируемость**: Легкость увеличения количества узлов и пропускной способности.
4. **Надёжность**: При отказе одного канала данные могут автоматически перенаправляться другим маршрутом.

<Callout>
В современных организациях компьютерные сети позволяют централизованно управлять ресурсами и обеспечивать безопасность данных.
</Callout>

## Пример сетевой команды

Ниже приведён простой пример использования команды `ping` в операционных системах семейства Unix/Linux для проверки доступности узла в сети:

```bash title="ping-example.sh"
// Проверяем доступность сайта example.com
ping -c 4 example.com
```

Данная команда отправляет четыре эхо-запроса (ICMP) и отображает статистику задержек при получении ответа.

# Коммуникационное оборудование локальных сетей

## Основные компоненты сетей  
**Коммутаторы** — ключевые устройства для управления трафиком в локальных сетях. Их задачи:  
- **Разделение домена коллизий**: Изоляция трафика между портами, предотвращение конфликтов данных.  
- **Адресная фильтрация**: Передача кадров только целевому устройству на основе MAC-адресов.  
- **Поддержка топологий**: Обеспечение связности в сложных сетях (например, с использованием VLAN или агрегации каналов).  

---

## Классификация коммутаторов  
### По уровню OSI:  
- **L2 (канальный уровень)**: Работают с MAC-адресами, поддерживают VLAN, STP, агрегацию каналов.  
- **L3 (сетевой уровень)**: Добавляют маршрутизацию на основе IP-адресов, сохраняя скорость коммутации.  

### По функционалу:  
- **Неуправляемые**: Простая передача данных без настройки (подходят для малых сетей).  
- **Управляемые**: Поддержка VLAN, QoS, безопасности (используются в корпоративных сетях).  

### По назначению:  
- **Уровень доступа**: Подключение конечных устройств (ПК, IP-телефонов).  
- **Уровень распределения**: Агрегация трафика между VLAN, применение политик безопасности.  
- **Уровень ядра**: Высокоскоростное соединение между сегментами сети.  

---

## Ключевые технологии  

### VLAN (IEEE 802.1Q)  
**Цель**: Создание изолированных сетевых сегментов в рамках одной физической инфраструктуры.  
- **Port-based VLAN**: Назначение VLAN по порту коммутатора.  
- **Tag-based VLAN**: Добавление метки VLAN в заголовок кадра для гибкой сегментации.

### Spanning Tree Protocol (STP/RSTP)  
**Задача**: Предотвращение петель в топологии сети.  
- **STP (IEEE 802.1D)**: Блокирует избыточные пути, создавая древовидную структуру.  
- **RSTP (IEEE 802.1W)**: Ускоренное восстановление связей при обрывах.

### Агрегация каналов (IEEE 802.3ad)  
Объединение нескольких физических линий в одну логическую для:  
- Повышения пропускной способности.  
- Обеспечения избыточности (при обрыве одного канала трафик перераспределяется).

### Power over Ethernet (PoE)  
Передача электропитания через кабель витой пары для устройств:  
- IP-телефонов, камер, точек доступа.  
- Стандарты: IEEE 802.3af (до 15.4 Вт), IEEE 802.3at (до 30 Вт).

## Уровни корпоративных сетей  

### Уровень доступа (Access Layer)  
- **Задачи**: Подключение пользователей, базовая безопасность (Port Security, DHCP Snooping).  
- **Оборудование**: Неуправляемые/управляемые коммутаторы с поддержкой PoE.  

### Уровень распределения (Distribution Layer)  
- **Задачи**: Маршрутизация между VLAN, применение списков контроля доступа (ACL), фильтрация трафика.  
- **Оборудование**: Управляемые L3 коммутаторы.  

### Уровень ядра (Core Layer)  
- **Задачи**: Обеспечение высокой пропускной способности и минимальных задержек.  
- **Оборудование**: Высокопроизводительные коммутаторы с поддержкой 10/40/100 Гбит/с.  

---

## Функции безопасности  
- **DHCP Snooping**: Защита от поддельных DHCP-серверов.  
- **Dynamic ARP Inspection**: Блокировка ARP-спуфинга.  
- **IP Source Guard**: Проверка подлинности IP-адресов.  
- **Port Security**: Ограничение числа MAC-адресов на порту.  

---

## Эволюция оборудования  
- **Концентраторы (Hub)**: Устаревшие устройства, создающие домен коллизий.  
- **Повторители (Repeater)**: Усиливают сигнал для увеличения расстояния передачи.  
- **Медиаконвертеры**: Преобразуют сигналы между медными и оптическими средами.  

> **Ключевые термины**: MAC-адрес, STP, VLAN, агрегация каналов, PoE, L3 коммутатор.
